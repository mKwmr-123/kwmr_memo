届け先が同じA社でも、部署と内線及び担当者が違う場合、その内の1箇所の部署と内線、担当者を取得するケース

SQL文

  1. SELECT 会社名
  2. ,部署
  3. ,担当者
  4. ,内線
  5. ,納品合計数
  6. FROM ( SELECT 会社名
  7. ,部署
  8. ,担当者
  9. ,内線
  10. ,SUM(納品数量) OVER (PARTITION BY 会社名) AS 納品合計数
  11. ,ROW_NUMBER() OVER (PARTITION BY 会社名 ORDER BY 届け先ID) AS グループNo
  12. FROM 届け先テーブル )
  13. WHERE グループNo = 1
  14. ORDER BY 会社名
